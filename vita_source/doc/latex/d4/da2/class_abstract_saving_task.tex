\doxysubsection{Detailed Description}
Abstract class part of an executer design pattern. Each time a tree is saved a set of executers will write the tree content. Different executers provide specific persistance of the tree, such as the generation of CCO files or VTK, depending if is a checkpoint for further tree generation or just a visualisation temporal file.
