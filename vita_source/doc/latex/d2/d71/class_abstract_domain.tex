\doxysubsection{Detailed Description}
Abstract domain class. Several domain methods guarantee the eficiency during the CCO generation, e.\+g., get\+Local\+Neighborhood and get\+DLim, which models the perfusion area of certain terminal in order to establish the minimum length criterion for the new vessel and the local set of vessels where it can be attached to the tree.
