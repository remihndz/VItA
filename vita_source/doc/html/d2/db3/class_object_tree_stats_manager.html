Detailed Description

Facade class used to access to all statistical analysis services implemented. The class aims to return vectors with the statistical results and no plots or any other outputs. Outputs are expected to be generated in the CCORendering library to decouple GUI from data and model components. The last is key to support the CCO library in clusters or environments with no GUI capabilities.
